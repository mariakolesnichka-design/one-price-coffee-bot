# One Price Coffee Bot

Telegram-бот для рецептур, сроков хранения и базы знаний One Price Coffee.

## Функции

- Меню по категориям
- Поиск напитка текстом
- Рецептуры и технологии приготовления
- Сроки хранения
- Избранное
- Несколько администраторов через ADMIN_IDS
- SQLite база данных

## Локальный запуск

1. Установить зависимости:

```bash
py -m pip install -r requirements.txt
```

2. Создать файл `.env` рядом с `bot.py`:

```env
BOT_TOKEN=НОВЫЙ_ТОКЕН_ОТ_BOTFATHER
ADMIN_IDS=123456789,987654321
```

3. Запустить:

```bash
py bot.py
```

## Важно

Токен нельзя публиковать в GitHub. Файл `.env` не загружайте в репозиторий.

## Как обновлять начинку

Рецепты лежат в `data/seed.json`. После изменения можно удалить файл `one_price_coffee.db` и запустить бота заново — база пересоздастся.
